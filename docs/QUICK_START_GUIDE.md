# Guia Rapido de Uso - MarketDash

## 1. Acesso
1. Abra a aplicacao e faca login.
2. Se nao tiver conta, use o cadastro.

## 2. Upload de CSV
1. Acesse `Dashboard > Upload`.
2. Selecione o arquivo CSV.
3. Aguarde a validacao e o processamento.
4. Confirme o dataset criado.

## 3. Dashboard
1. Acesse `Dashboard`.
2. Verifique KPIs principais (receita, custo, lucro).
3. Use filtros por data, produto e valores.

## 4. Investimentos (Ad Spends)
1. Acesse `Dashboard > Investimentos`.
2. Adicione gastos manualmente ou importe em lote.
3. Revise os valores aplicados ao dataset.

## 5. Relatorios
1. Acesse `Dashboard > Relatorios`.
2. Aplique filtros conforme o periodo desejado.
3. Exporte CSV ou PDF quando necessario.

## 6. Dicas rapidas
- Use filtros para comparar periodos.
- Mantenha os gastos de anuncios atualizados.
- Exporte relatorios para compartilhar com o time.
